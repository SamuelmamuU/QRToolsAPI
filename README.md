# QRToolsAPI
api para conectar el sistema


## Ejecutarlo en Windows

### Crear el virtual enviroment
```bash
python -m venv vapi
```

### Activar el virtual enviroment
```bash
.\vapi\Scripts\activate
```
### Corre el codigo
```bash
python3 API.py
```

### Desactivar el virtual enviroment
```bash
deactivate
```
